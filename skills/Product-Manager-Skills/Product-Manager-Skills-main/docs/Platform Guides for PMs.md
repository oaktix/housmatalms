# Platform Guides for PMs

This page helps you pick the easiest setup path for your comfort level.

Part of the v0.65 onboarding update: clearer install/integration paths for PM teams.

If you are non-technical, start here first:
- [`Using PM Skills 101.md`](Using%20PM%20Skills%20101.md)

## Quick Pick

- "I want simple chat setup" -> [`Claude Desktop`](Using%20PM%20Skills%20with%20Claude%20Desktop.md) or [`ChatGPT Desktop`](Using%20PM%20Skills%20with%20ChatGPT%20Desktop.md)
- "I can use terminal commands" -> [`Claude Code`](Using%20PM%20Skills%20with%20Claude%20Code.md), [`Claude /slash commands`](Using%20PM%20Skills%20with%20Slash%20Commands%20101.md), or [`Codex`](Using%20PM%20Skills%20with%20Codex.md)
- "I need automation" -> [`n8n`](Using%20PM%20Skills%20with%20n8n.md) or [`LangFlow`](Using%20PM%20Skills%20with%20LangFlow.md)
- "I’m testing alternative stacks" -> [`OpenClaw`](Using%20PM%20Skills%20with%20OpenClaw.md)
- "I’m using Anthropic workspace tools" -> [`Claude Cowork`](Using%20PM%20Skills%20with%20Claude%20Cowork.md)
- "My team is building custom agents" -> [`Python agents`](Using%20PM%20Skills%20with%20Python%20Agents.md)
- "I use AI coding copilots" -> [`Cursor`](Using%20PM%20Skills%20with%20Cursor.md) or [`Windsurf`](Using%20PM%20Skills%20with%20Windsurf.md)
- "I use app/prototype builders" -> [`Bolt`](Using%20PM%20Skills%20with%20Bolt.md) or [`Replit Agent`](Using%20PM%20Skills%20with%20Replit%20Agent.md)
- "I use enterprise workflow builders" -> [`Make.com`](Using%20PM%20Skills%20with%20Make.com.md)
- "I use autonomous engineering agents" -> [`Devin`](Using%20PM%20Skills%20with%20Devin.md)
- "I run multi-agent orchestration" -> [`CrewAI`](Using%20PM%20Skills%20with%20CrewAI.md)
- "I use Google's ecosystem" -> [`Gemini`](Using%20PM%20Skills%20with%20Gemini.md)
